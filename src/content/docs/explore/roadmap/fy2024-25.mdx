## Completed Features (1.2.0-RC)

| Feature | Description | Status | Release |
|---------|-------------|--------|---------|
| 📦 **Connector Framework** | Add and manage connectors via APIs. Create and drop streams (Flink) or batch jobs (Spark) via APIs. Add and manage custom jobs via APIs. **Enhanced stability** with rigorous testing and fixes for key issues. Open-source connectors for **Kafka**, **JDBC**, and **Object Store**. | ✅ Completed | 1.2.0-RC |
| 🔌 **Open-Source Connectors** | Supported connectors: **Kafka**, **JDBC**, **Object Store** | ✅ Completed | 1.2.0-RC |
| 🏞️ **Lakehouse** | Transactional Data Lake Platform for scalable and reliable data processing and storage | ✅ Completed | 1.2.0-RC |
| ⚙️ **Automation Refactor** | Stabilized scripts for **greater reliability** and ease of use. Support for **AWS**, **GCP**, **Azure**, and on-premises environments. Streamlined installation for seamless deployments across any setup. | ✅ Completed | 1.2.0-RC |
| 📊 **Dataset Management** | Comprehensive dataset management with automatic schema evolution, seamless import/export, auto schema generation and indexing, proactive alerts and monitoring, masking and encryption, JSONata and SQL transformations, data exhausts, and data aliases for simplified replay and migration without downtime. | ✅ Completed | 1.2.0-RC |
| 🛡️ **RBAC** | Manage users and roles via APIs for robust role-based access control | ✅ Completed | 1.2.0-RC |
| 📡 **Obsrv Exporter** | Export OpenTelemetry-compliant monitoring data for seamless integration with external monitoring systems | 🔄 Partially Completed | 1.2.0-RC |
| 🖥️ **Obsrv Management Console** | Full-featured console for managing datasets and connectors. Effortless dataset lifecycle management (Draft → Ready for Publish → Published → Retired). Source Connector Configuration for Kafka, JDBC, Object Store. Seamless Export/Import, Automatic Schema Generation & Indexing, Proactive Alerts & Monitoring, and Resource Monitoring. | ✅ Completed | 1.2.0-RC |
| 📝 **Query APIs** | Query from multiple data sources such as **Druid**, **Hudi**, **Lakehouse**. Exhaust API for retrieving data exhausts from blob stores. Template API for creating and using reusable query templates across datasets and environments. | ✅ Completed | 1.2.0-RC |
