## Obsrv 1.2.0 Release

**📅 Release Date:** *November 8, 2024*

---

### 🌟 What's New in Obsrv 1.2.0 RC?

Obsrv 1.2.0 RC is here, and it's all about **stability, performance,** and introducing the powerful new **Obsrv Management Console**. 🚀

---

#### 🚀 Core Innovations

**🛠️ Transaction Data Lake Platform**

- Major stabilization improvements for seamless integration with Obsrv.
- Effortlessly ingest and manage data in the transaction data lake platform.

**🔗 Data Connector Framework**

- **Enhanced stability** with rigorous testing and fixes for key issues.
- Now featuring **open-source connectors** for:
  - **Kafka**
  - **JDBC**
  - **Object Store**

:::note
Sink connectors are not available in this release.
:::

---

#### 🌐 API Enhancements

Experience robust and reliable API performance with:

- **Dataset CRUD operations**
- **Export/Import workflows**
- **Status transitions**
- **Schema generation**
- **Template query APIs**
- **Data ingestion and querying**

---

#### 🔧 Obsrv Management Console

Say hello to the all-new **Obsrv Management Console** — your one-stop solution for managing datasets and connectors.

**🌟 Key Features:**

1. **Effortless Dataset Management** — Manage datasets across their lifecycle: **Draft → Ready for Publish → Published → Retired**.
2. **Source Connector Configuration** — Configure **Kafka**, **JDBC**, and **Object Store** connectors directly from the console.
3. **Seamless Export & Import** — Transfer datasets between environments with robust Export and Import functionalities.
4. **Automatic Schema Generation & Indexing** — Instantly generate schemas and effortlessly index data from **Redis**, **Druid**, and **Lakehouse**.
5. **Proactive Alerts & Monitoring** — Set up intelligent alerts to monitor dataset health and ensure smooth operations.
6. **Resource Monitoring** — Keep tabs on system performance with integrated resource monitoring tools.

---

#### 🔄 Automation Enhancements

**⚙️ Refined Automation Framework**

- Stabilized scripts for **greater reliability** and ease of use.
- Broadened support for:
  - **Cloud platforms:** AWS, GCP, Azure
  - **On-premises environments**
- Streamlined installation for smooth deployments across any setup.

---

#### ⚠️ Deprecation Notice

- **Version 1.x APIs** are now **deprecated** and unsupported.
- Transition to **V2 APIs** to access the latest features and ensure continued support.

---

#### 📖 Migration Guide

A comprehensive guide to help you migrate from **v1.0.x to v1.2.0 RC** will be available soon.

This release offers unparalleled **stability**, **performance improvements**, and the all-new **Obsrv Management Console**. Upgrade now to unlock a new era of data management efficiency.
