## Obsrv 1.2.0-RC Release

**📅 Release Date:** _November 8, 2024_

---

### 🌟 **What's New in Obsrv 1.2.0 RC?**

Obsrv 1.2.0 RC is here, and it's all about **stability, performance,** and introducing the powerful new **Obsrv Management Console**. Get ready for a redefined data management experience. 🚀

---

#### 🚀 **Core Innovations**

##### 🛠️ **Transaction Data Lake Platform**

- Major stabilization improvements for seamless integration with Obsrv.
- Effortlessly ingest and manage data in the transaction data lake platform.

##### 🔗 **Data Connector Framework**

- **Enhanced stability** with rigorous testing and fixes for key issues.
- Now featuring **open-source connectors** for:
  - **Kafka**
  - **JDBC**
  - **Object Store**

💡 _Note: Sink connectors are not available in this release._

---

#### 🌐 **API Enhancements**

Experience robust and reliable API performance with:

- **Dataset CRUD operations**
- **Export/Import workflows**
- **Status transitions**
- **Schema generation**
- **Template query APIs**
- **Data ingestion and querying**

---

#### 🔧 **Obsrv Management Console**

Key Features:

1. **Effortless Dataset Management** — Manage datasets across their lifecycle: Draft → Ready for Publish → Published → Retired.
2. **Source Connector Configuration** — Configure Kafka, JDBC, and Object Store connectors directly from the console.
3. **Seamless Export & Import** — Transfer datasets between environments.
4. **Automatic Schema Generation & Indexing** — Instantly generate schemas and index data from Redis, Druid, and Lakehouse.
5. **Proactive Alerts & Monitoring** — Set up intelligent alerts to monitor dataset health.
6. **Resource Monitoring** — Integrated system performance monitoring.

---

#### ⚠️ **Deprecation Notice**

- **Version 1.x APIs** are now deprecated and unsupported. Transition to **V2 APIs** to access the latest features.

---

## Obsrv 1.1.0 Beta Release

**📅 Release Date:** _31st August 2024_

---

### 🌟 **What's New?**

#### 🚀 **Core Innovations**

- **Transaction Data Lake Platform** — Supercharged with Apache Hudi (v0.15.0).
- **Data Connector Framework** — Easy connector setup, flexible framework, batch and stream connectors. New source connectors: Kafka, Object Store, JDBC, Knowlg.
- **Real-Time Data Transformation** — Encryption, data masking, and JSONata transformations.
- **Cache Indexing** — Redis Cache indexing via a streaming Flink job for faster denormalization.
- **Schema Validation Enhancement** — Upgraded validator supporting Draft-12 of the JSON Schema specification.
- **Enhanced Security** — Fixes for critical security vulnerabilities including SQL injection.

#### 🌐 **API Enhancements**

- **Dataset Management APIs** — Full CRUD, status transitions, health monitoring, export/import, copy/reset, schema generation, dataset listing.
- **Connector APIs** — Read and list connectors.
- **Schema and Event Validation API** — Validate schemas and events against established standards.
- **Template APIs** — Full CRUD for templates, plus template query API.
- **Data APIs** — Data query (SQL/native), data ingestion, data exhaust.
- **Alert and Notification APIs** — Custom alerts, notification channels (Slack, email, Discord).

#### 🔧 **Obsrv Management Console**

Full-featured console for dataset and connector lifecycle management, export/import, automatic schema generation, proactive alerts, resource monitoring, and data querying.

#### 🔧 **Automation Upgrades**

- Expanded support for AWS, GCP, Azure, and on-premises environments.
- Simplified installation scripts.

#### 🚫 **Deprecation Notice**

APIs from version 1.0.x are no longer supported. Transition to the new APIs.
