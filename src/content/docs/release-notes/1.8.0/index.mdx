**Release Date:** 2025-May-01

**Release Type:** Minor

**Focus:** Infra Monitoring stability, AWS Kubernetes upgrade, Documentation and Query Auditing.

| # | Category | Title | Summary |
|---|---|---|---|
| 1 | 🛠️ Improvements | Improved Alerting & Notification | Enhanced Obsrv alerting system to provide clearer, more actionable alerts for easier debugging and operational response. |
| 2 | 🔄 Upgrade/Changes | AWS EKS Cluster Upgrade | Upgraded the AWS EKS cluster to version 1.31.0 from 1.30.0. |
| 3 | 📚 Documentation | Alerting Documentation Hosted | Available for reference by Ops and Dev teams. |
| 4 | ✨ New Feature | Query Audit Logs | Added audit logging to capture "who", "what", and "when" for all executed queries. Default, it's disabled and can be enabled during the installation. |
