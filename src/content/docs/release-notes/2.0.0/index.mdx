**Release Date:** 2025-08-13

**Release Type:** Major

**Focus:** Platform-wide software upgrades, Kafka 4.0 migration support, and stability improvements.

---

## 1. Overview

The **Obsrv 2.0.0** release delivers major platform upgrades across core data processing and query components. This release ensures compatibility with **Kafka 4.0** and introduces performance, stability, and maintainability improvements. It also includes critical bug fixes and enhanced data migration handling from **Kafka 3.6 to 4.0**.

## 3. Major Changes

### 3.1 Kafka 4.0 Migration Support

- Added full platform compatibility for Kafka 4.0.
- All dependent services updated (Flink, Secor, Druid, Connectors).
- Data migration handled via MirrorMaker 2 (MM2) for Kafka 3.6 → 4.0 cutovers.

### 3.2 Connectors 2.0

- Flink runtime upgraded to 1.20.
- Kafka client upgraded for 4.0 compatibility.

### 3.3 Software Modernization

- All major core components upgraded to the latest stable versions.
- Updated Docker base images and dependencies for security compliance.

### 3.4 Bug Fixes & Stability

- Fixed checkpoint instability in Flink jobs under high throughput.
- Resolved occasional ingestion stalls in Secor when Kafka rebalance occurs.
- Improved Druid Kafka indexing task reliability.
