**Release Date:** 2025-12-19

**Release Type:** Minor

**Objective:** Upgrade the Obsrv web console from React version 19.0.0 to 19.0.1 to fix a security vulnerability present in version 19.0.0.

:::note
a. While upgrading to **release 2.1.3**, please use the **2.0.2** version of **enterprise-automation/obsrv-scripts-infy**.

b. While upgrading from previous releases of **v2.1.x to v2.1.3**, upgrade only the **en-bundles**.
:::
