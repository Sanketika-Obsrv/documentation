**Release Date:** 2025-09-23

**Release Type:** Minor

**Focus:** Migration of deprecated Bitnami container images to the sanketikahub repository to ensure stability, maintainability, and long-term support.

---

## 1. Overview

The Obsrv 2.1.1 release standardizes container images across all deployments by migrating deprecated Bitnami and Bitnamilegacy images to the sanketikahub repository. This update ensures consistent image availability, simplifies future automation workflows, and improves operational reliability for Obsrv deployments.
