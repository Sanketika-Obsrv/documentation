**Release Date:** 2026-06-08

**Release Type:** Minor

**Objective:** Added required Obsrv Superset configuration for embedding support.

**Focus:** Added `EMBEDDED_SUPERSET` in the feature flags and related settings to support embedding Superset dashboards in external applications.

:::note
- When upgrading to release **2.1.6**, use version **2.0.3** of **enterprise-automation/obsrv-scripts-infy**.
- For upgrades from **v2.1.x to v2.1.6**, upgrade only the **core-infra** bundle.
:::
